Эта программа была написана для быстрой генерации схожего текста. К примеру с его помощью можно быстро создать 100 case или для вывода информации как при многоступенчатом цикле.

К примеру нужно создать 100 case:

Пишем шаблон:

case |&1&|:
{
break;
}

И указываем в диапазонах сколько case нам нужно сделать. На месте |&1&| будет вставляться значение из диапазона. Диапазон работает как цикл for.

Результат при диапазоне 0   10:

case 0:
{
break;
}
case 1:
{
break;
}
case 2:
{
break;
}
case 3:
{
break;
}
case 4:
{
break;
}
case 5:
{
break;
}
case 6:
{
break;
}
case 7:
{
break;
}
case 8:
{
break;
}
case 9:
{
break;
}


Или к примеру нам нужно заполнить двумерный массив вручную обращаясь напрямую к каждому элементу:

Пишем шаблон:

array[|&1&|][|&2&|] = |&1&| + |&2&|;

Результат при диапазоне 0 до 5 и от 0 до 3(Диапазоны отделяются пробелом. В textbox от будет "0 0", в textbox до будет "5 3"):

array[0][0] = 0 + 0;

array[0][1] = 0 + 1;

array[0][2] = 0 + 2;

array[1][0] = 1 + 0;

array[1][1] = 1 + 1;

array[1][2] = 1 + 2;

array[2][0] = 2 + 0;

array[2][1] = 2 + 1;

array[2][2] = 2 + 2;

array[3][0] = 3 + 0;

array[3][1] = 3 + 1;

array[3][2] = 3 + 2;

array[4][0] = 4 + 0;

array[4][1] = 4 + 1;

array[4][2] = 4 + 2;

Если нам нужно вставить строку выбранную случайно из массива строк, то нужно применить такой шаблон:

Пример |&1&|: |&0&|;

Результат при диапазоне 0 до 5 и стоках "create break continue case switch c#" в textbox "Диапазон для случайных вставок":

Пример 0: case;

Пример 1: continue;

Пример 2: create;

Пример 3: c#;

Пример 4: create;

На место |&0&| будет вставлено случайное слово из строки "create break continue case switch c#" (Слова разделяются пробелом!).


Ну и если поставить галочку в checkbox , то вместо чисел из диапазона будет вставляться слово из "Диапазон для случайных вставок" в соответствии с его порядковым номером.

Пишем шаблон:

Пример |&1&|: |&2&|;

Результат при диапазоне 0 до 5 и стоках "create break continue case switch c#" в textbox "Диапазон для случайных вставок":

Пример create: break;

Пример create: break;

Пример create: break;

Пример create: break;

Пример create: break;



Для быстрого вызова |&&| в которых нужно написать номер цикла к которому обращаемся нужно нажать ctrl+q.

Для удобства при двойном нажатии в RichTextBox его содержимое переносится в буфер.
